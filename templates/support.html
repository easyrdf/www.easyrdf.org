Support

If you have any general questions or need help using EasyRdf, then there are a couple of forums for you to get help:

Mailing List
easyrdf@googlegroups.com on Google Groups
IRC
#easyrdf on Freenode

Issue tracker

Please submit bug reports and enhancement requests to the issue tracker on GitHub:

http://github.com/njh/easyrdf/issues
I would be delighted if you could supply a pull request or patch with bug reports but any major enhancements or changes should be discussed with me first, as I cannot promise to accept all pull requests.

Security Issues

If you think that you have found a security issue in EasyRdf, please don't use the mailing list or the issue tracker. Instead, all security issues must be sent to security [at] easyrdf.org. Emails sent to this address are forwarded to the development team to look at.
